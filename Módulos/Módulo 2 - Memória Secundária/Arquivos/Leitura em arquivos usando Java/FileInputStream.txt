Como você viu, a classe usada para a leitura de dados em um arquivo é classe FileInputStream (Links para um site externo.), mas que também só oferece um método básico: o método read(). Para que tenhamos métodos para de leitura de tipos primitivos, precisaremos também do apoio de uma outra classe. Dessa vez, será a classe DataInputStream, que oferece métodos como readInt() para a leitura de números inteiros, readFloat() para a leitura de números reais e readUTF() para a leitura de strings.
