A classe FileOutputStream nos oferece apenas o método write() para escrevermos dados no arquivo. Esse método pode escrever apenas 1 único byte ou um vetor de bytes. Assim, precisamos do apoio de uma segunda classe que nos ofereça métodos para a escrita dos tipos primitivos (int, float, boolean, etc.). Essa classe é a DataOutputStream. Ela contém métodos como writeInt() para a escrita de números inteiros, writeFloat() para a escrita de números reais, writeUTF() para a escrita de strings e muito mais.
